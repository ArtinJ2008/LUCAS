# Ueda et al. (2021) komatiite experiment reference data

This directory preserves the primary data used for LUCAS's first laboratory
reference reconstruction. The source is Ueda et al.'s Mendeley Data v4 record,
DOI [`10.17632/dr9kxs8yc8.4`](https://doi.org/10.17632/dr9kxs8yc8.4), published
under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). The associated
paper is Ueda et al. (2021), *Geochemistry, Geophysics, Geosystems*, DOI
[`10.1029/2021GC009827`](https://doi.org/10.1029/2021GC009827).

## Contents and integrity

- `raw/` contains byte-for-byte copies of the four v4 source workbooks.
- `source_manifest.toml` records source file identifiers, sizes, and SHA-256
  hashes for all four Mendeley workbooks plus the versioned normalized Table 2
  CSV. Local hashes and the exact expected path inventory must match before use.
- `fluid_time_series.csv` is a machine-readable adaptation of source Table 2.
  Values are not rounded or interpolated. Column names were normalized, merged
  experiment labels were filled down, and `n.a.`/`n.d.` were split into empty
  value cells plus explicit status columns. These are the only changes.

Table 3 compiles measurements originally reported by third parties. LUCAS keeps
the source workbook for auditability but does not redistribute a transformed
Table 3 or use it as a calibration target until each original citation and its
terms have been reviewed.

## What “reproduction” means at this stage

The implemented benchmark reconstructs the published sampling series and
checks the byte-pinned source and normalized artifacts plus selected semantic
invariants. It does **not** claim to predict the
water-rock reactions. A predictive reproduction still requires an independently
reviewed activity model, aqueous/mineral thermodynamic database, mineral surface
areas, kinetic rate laws, and a discrete-withdrawal model of the authors'
flexible-gold-bag batch reactor.

The experimental observations can be used as upstream source-fluid constraints
or later validation targets. They must not be treated as direct observations of
Hadean seawater, a natural vent, or life's formation.
